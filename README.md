# PIPIN
####Blue Server
######Aplicacion realizada en Java que funciona como servidor para la aplicacion cliente Blue.

> El Wiki se encuentra en construccion paralela con el proyecto.

Modulos de Servicio Completos:
* MySQL
* HTTP Server (Texto e Imagenes)
* Subir Imagenes (Base64 a .png/.jpg)
* Envío de correos (para modulo API /forgot)

Modulos API Completos:
* /login
* /register
* /addKid
* /notifPoll
* /shadowPoll
* /kidPoll
* /getKidList
* /kidData
* /editShadow
* /forgot
* /editKid
* /addContact
* /editContact
* /deleteContact

Modulos API Pendientes:
* /postChat
* /sendPic
* /alertParent

>Folders necesarios:
* C:\avatars
* C:\avatars\contacts
* C:\avatars\kid
* C:\avatars\parents
* C:\pics
